Essential Git Commands After Changing Code
git init                             # This is usually the first command you'll run in a new project
git status                           # Check which files have changed and see all the track and untrack files
git add .                            # Stage all modified files within the main folder
git reset                            # Unstages all changes
git add sample.html                  # Stage specific modified files
git rm --cached sample.html          # Unstages a specific file
git commit -m "Your commit message"  # Commit the changes
git push origin main                 # Push the changes to GitHub

# USING GIT COMMIT FOR MULTIPLE NOTES YOU WANNA SAY
git commit                           # Commit the changes
(SHOW THE FILE FOR MESSAGE)          # You can type now your message here
(Esc + :q! + Enter)                  # This will force quit without saving
(Esc + :wq + Enter)                  # This will save and quit
